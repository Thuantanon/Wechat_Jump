# 微信跳一跳的辅助的Java实现
手机启动开发者模式并连接到电脑。
由鼠标操作，在起点按下终点释放，辅助程序计算跳跃时间。
仅供学习参考：
没做机型适配，不同分辨率手机可能有偏差，测试手机为720*1280。
